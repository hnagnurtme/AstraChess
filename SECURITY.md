# Security Policy

## Supported Versions

We support security updates for the following versions of AstraChess:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅ Yes    |
| < 1.0.0 | ❌ No     |

## Reporting a Vulnerability

We take the security of AstraChess seriously. If you find any security vulnerabilities, please do not report them publicly via GitHub issues. Instead, please report them responsibly by sending an email to the project maintainers.

We will acknowledge your report within 48 hours and provide a detailed timeline for addressing the issue. Once resolved, we will publish a security advisory and credit you for your responsible disclosure.
